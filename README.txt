Starting the local database/frontend

1. Open two terminals
2backend. Move to invoicify
2frontend. Move to invoicify-frontend
3backend. Run ./mvnw spring-boot:run
3frontend. Run npm start
4. Connect to localhost:4200
5. login is USER: admin PASSWORD: admin by default


Note: Change the password in the src/main/resources/application.properties.default
then run the command "cp application.properties.default application.properties"
